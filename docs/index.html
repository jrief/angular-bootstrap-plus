What is Angular Bootstrap Plus?

This is a collection of Angular directives for Bootstrap, which neither made it into Angular Strap, nor UI Bootstrap.

Currently there is only one directive <aw-select>...</aw-select>, but a compatible slider plugin has already been prepared.

The alert reader might note, that a similar directive exists in Angular Strap, but here one must pass a list of configuration options to this directive. Since I had the need to style the internals of the drop down menu using pure HTML, this directive didn't fit.

In UI Bootstrap no such directive exists and after offering this module to the maintainers, it wasn't accepted. Appearently the UI Bootstrap team wants to follow the same path as the Angular Strap team. My intention however is to use as much declarative HTML instead of configuration objects.
